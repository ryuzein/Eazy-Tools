## Installation

### Windows or Linux
You can download the latest composer in [here](https://getcomposer.org/download/).

    git clone https://github.com/warifp/SpotifyToolkit

    composer 
    
### Termux

    pkg install php

    pkg install curl

    pkg update

    git clone https://github.com/warifp/SpotifyToolkit
